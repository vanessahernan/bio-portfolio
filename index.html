01. Sobre mí

¡Hola! Me llamo Vanessa y disfruto creando cosas que viven en internet. Mi interés en el desarrollo web comenzó en 2024, cuando decidí que quería transformar ideas en realidades interactivas.

Hoy en día, tengo el privilegio de trabajar como desarrolladora freelance, ayudando a clientes a llevar sus visiones a la web con HTML, CSS, JavaScript y Vue.js.

Mi objetivo es siempre construir productos que no solo cumplan con los requisitos, sino que superen las expectativas en rendimiento y diseño.

Aquí hay algunas tecnologías con las que trabajo actualmente:

  • JavaScript (ES6+)
  • Vue.js
  • HTML & CSS
  • Git & GitHub
  • Python (basico)
